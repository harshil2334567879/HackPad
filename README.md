# HackPad

Hi! This is my first custom macropad that I made for the Hack Club HackPad program.

I designed the PCB myself in KiCad and learned how to make a schematic, assign footprints, route the PCB, add copper pours, fix DRC errors, and generate the manufacturing files. Before this project I had never designed a PCB, so this was a great learning experience.

## Features

- 6 Mechanical Keys
- Rotary Encoder
- 0.91" OLED Display
- 7 SK6812 MINI-E RGB LEDs
- Seeed Studio XIAO RP2040
- Fully programmable with KMK/QMK

## Why I Made It

I wanted to learn PCB design and build something that I could actually use on my desk. Instead of buying a macropad, I thought it would be much more fun to make one from scratch.

This project helped me understand how electronic components are connected, how PCB routing works, and how a custom keyboard is designed.

## Challenges

The hardest part was routing the PCB and fitting everything in a compact layout. I had to move components around multiple times before I got a clean design. Understanding footprints and PCB rules was also confusing at first, but I figured it out while working on the project.

## What I Learned

- KiCad basics
- PCB routing
- Schematics
- Copper pours
- Design Rule Check (DRC)
- PCB manufacturing process
